Hyaenidae Distribution Map for Africa

Visualizing ranges for the four extant species.

Created by:
Benjamin Tjepkes

Last updated:
November 8, 2023

This map shows the current ranges for the four extant species in the family Hyaenidae across the African continent.

Task & Objectives

The prompt for this map was for Day Eight of the 2023 30 Day Map Challenge, which designated ‚ÄúAfrica‚Äù for the main subject. Hyenas have always fascinated me with their complex behavior, clan structure, and hunting abilities. This prompt was a perfect time to explore their ranges!

Tools & Workflow

Spatial data were sourced from the IUCN for each species. Map was produced in ArcGIS Pro 3.2.

The comprehensive layout options in ArcGIS Pro have always helped me out, but this time it was very simple to link the extents of each sub-map to ensure perfect alignment.
